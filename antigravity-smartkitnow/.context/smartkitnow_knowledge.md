# SmartKitNow Project Context

## Site Overview

SmartKitNow.com is a free online calculator tools website offering 739 calculators across 15 categories. The site targets users searching for specific calculation tools across sports, finance, health, pets, and more.

## Technical Stack

- Frontend: Modern web application (JavaScript-based)
- URL Structure: `https://www.smartkitnow.com/[category]/[calculator-slug]`
- Sitemap: Available at `/sitemap.xml`

## Content Guidelines

### Title Format
`[Calculator Name] Calculator 2026 | Free [Category] Tool`

Maximum 60 characters. Include:
- Primary keyword at start
- Year for freshness
- "Free" or benefit word
- Category context

### Meta Description Format
Maximum 155 characters. Include:
- Primary keyword naturally
- Clear benefit statement
- Call to action
- Unique value proposition

### Schema Requirements
Every calculator page needs:
1. WebApplication schema (type, name, description, offers)
2. FAQPage schema (3-5 relevant questions)
3. BreadcrumbList schema (navigation path)
